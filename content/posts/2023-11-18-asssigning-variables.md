This is how you assign a value from right to left.

```perl
my $x = 1;
```

this is how you assign a value from left to right.

# Perl

```perl
do_something(12, sub ($x) { ... });
```

# JavaScript

```js
do_something(1, function(x) { ... })
```

# F#

```fsharp
doSomething 1 (fun x -> ... )
```
